    2019年10月28日

    《青少年编程能力等级》标准发布，包含151个编程能力要求

    近日，《青少年编程能力等级》标准联合发布会在北京清华大学举行。据悉，该系列标准主要针对课外培训领域，发布会公示了《青少年编程能力等级》标准第1部分「图形化编程」标准和第2部分「Python编程」标准。

    「图形化编程」标准将分为三级，「Python编程」标准将分为四级，两个等级标准分别包含103个图形化能力要求和48个Python能力要求。第一次青少年编程能力等级测试将于2019年年底前于部分地区落地，预计明年将开放全国范围的申报服务。第3部分机器人编程和第4部分C++编程将启动研究。（芥末堆）

    史玉柱：区块链会深刻改变社会和公司管理架构

    史玉柱在微博发表点评称，区块链的技术攻关难度不太大，难在深刻理解区块链思想，并成功应用于各场景。区块链会深刻改变社会，改变公司管理架构。以前两陌生人间做交易，彼此缺乏信用，不得不找双方都信任的第三方（银行、支付宝等）做中介，去完成交易。区块链能有效解决两陌生人间的信用问题，提高交易效率，改变交易流程。（新浪财经）

    北京5G基站数量将超1.4万个，明年全市5G网络全覆盖

    北京市经济和信息化局与顺义区人民政府举办的“5G-50”技术与应用专家论坛上获悉，预计今年年底，北京全市5G基站将超1.4万个，基本实现五环内部署完成，到2020年实现城郊5G网络全覆盖，为5G场景示范应用提供良好的网络环境保障。截至目前，三大运营商在全市共建成1.1万个5G基站，5G网络信号可覆盖六环内人口密集区域、大兴国际机场等区域。（北京日报）

    腾讯安全：首次攻破超声波屏下指纹识别技术

    在近日召开的GeekPwn 2019国际安全极客大赛上，腾讯安全玄武实验室披露了指纹识别领域的最新研究——自动化破解多种类型指纹识别。腾讯方面表示，该研究通过提取用户在日常生活中留存的指纹，自动化进行克隆复原，进而通过各种指纹设备的验证。这次挑战也是国际上第一次成功攻破超声波屏下指纹识别技术。（新浪科技）

    微软赢得五角大楼的100亿美元JEDI云合同

    微信支付「向手机号转账」功能正式上线

    微信支付正式上线新功能「向手机号转账」。该功能意味着在不是微信好友，不知道对方微信号，不能面对面扫码等情况下，通过手机号就能发起支付。用户使用该功能前，需要先点击微信页面右上角的「+」-「收付款」-「向银行卡或手机号转账」-「手机号收款设置」，开启「允许通过手机号向我转账」。如果你是转账方，点击「向手机号转账」，输入对方手机号即可。

    Netflix 开源多语言记事本环境 Polynote

    Netflix 的技术团队宣布开源多语言记事本环境 Polynote，源代码发布在 GitHub 上（https://github.com/polynote/polynote/blob/master/LICENSE），采用 Apache License 2.0 许可证。Polynote 支持 Scala、Python、SQL 和 Vega。开发者称，Polynote 向数据科学家和机器学习研究员提供了一个记事本环境，允许自由无缝的整合基于 JVM 的机器学习平台。Polynote 受到了集成开发环境（IDE）的启发，提供了类似 IDE 的功能如交互式自动完成、参数提示、内联错误高亮、支持 LaTeX 的富文本编辑器。
